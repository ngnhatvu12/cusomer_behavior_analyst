Customer Behavior Analyst
Overview

This project analyzes customer behavior using Python, PostgreSQL, and Power BI.
The process includes data loading, EDA, data cleaning, SQL analysis, dashboard creation, and reporting.

Dataset

The dataset contains:

Customer information
Purchase history
Product data
Sales records
Tools
Python (Pandas, Matplotlib, Seaborn)
PostgreSQL
Power BI
Jupyter Notebook
Project Steps
1. Load Data
Import dataset with Pandas
Check data types and missing values
2. EDA
Analyze sales trends
Explore customer behavior
Find top products and customers
3. Data Cleaning
Remove duplicates
Handle missing values
Format data properly
4. PostgreSQL
Store cleaned data
Run SQL queries for analysis
5. Power BI Dashboard
Create interactive visualizations
Build KPIs and charts
6. Report
Summarize insights and results
Dashboard

The dashboard includes:

Sales Overview
Revenue Trends
Customer Analysis
Product Performance
Results

Main outcomes:

Better understanding of customer behavior
Sales and revenue insights
Interactive dashboard for business analysis
